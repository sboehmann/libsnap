libsnap is an Open Source software project that implements the S.N.A.P networking protocol.

S.N.A.P is an free and open network protocol developed by High Tech Horizon. The S.N.A.P protocol is a generic and flexible network protocol designed to be useable in any type of applications where an easy to learn and light weighted network protocol is needed.
